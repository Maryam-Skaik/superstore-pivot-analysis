# Superstore Pivot Table Analysis (Excel)

## 📊 Project Overview

This project analyzes retail sales data using Excel Pivot Tables.  
The goal is to explore business performance across multiple dimensions such as region, product category, customer segment, and time.

The dataset used is the well-known Superstore dataset from Kaggle.

---

## 📁 Dataset

- Source: Kaggle Superstore Dataset  
- Contains information about:
  - Orders
  - Sales
  - Profit
  - Customers
  - Product categories
  - Regions and dates

---

## 🛠 Tools Used

- Microsoft Excel
- Pivot Tables

---

## 📊 Analysis Performed

The analysis is divided into four main perspectives:

### 1. Sales Overview
- Total sales per region
- Average profit per region

### 2. Product Analysis
- Sales by category and sub-category
- Top-performing products

### 3. Customer Analysis
- Sales by customer segment
- Top customers by total purchases

### 4. Time Analysis
- Monthly sales trends
- Yearly performance comparison

---

## 🔍 Key Insights

Detailed insights are documented in `insights.md`.  
Main findings include:

- Sales performance varies significantly across regions
- Certain product categories dominate overall revenue
- Consumer segment contributes the highest sales share
- Sales peak during Q4 months (seasonal trend)
